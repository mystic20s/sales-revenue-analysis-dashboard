# 📊 Sales & Revenue Analysis Dashboard

## Overview
This project presents an interactive Sales & Revenue Analysis Dashboard developed in Microsoft Excel. The dashboard helps analyze sales performance through KPIs, charts, and summary statistics to support business decision-making.

## Features
- Total Revenue KPI
- Total Units Sold KPI
- Average Order Value
- Revenue by Region Analysis
- Interactive Charts
- Clean and organized dashboard layout

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Formulas
- Slicers (if used)

## Dataset
A sample sales dataset containing:
- Order ID
- Order Date
- Product
- Region
- Units Sold
- Revenue

## Key Insights
- Regional revenue comparison
- Overall revenue performance
- Average order value
- Sales distribution
